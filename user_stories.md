# User Stories – Food Review Social Media App

## Customer User Stories

- As a user, I want to register so that I can write reviews.
- As a user, I want to log in securely.
- As a user, I want to add a restaurant.
- As a user, I want to write a review.
- As a user, I want to rate food experience.
- As a user, I want to view reviews from others.
- As a user, I want to comment on reviews.
- As a user, I want to like reviews.
- As a user, I want to view user profiles.
- As a user, I want to edit my review.
- As a user, I want to delete my review.
- As a user, I want to logout safely.

## Restaurant Owner User Stories

- As a restaurant owner, I want to create an account so that I can manage my restaurant.
- As a restaurant owner, I want to log in securely.
- As a restaurant owner, I want to create a restaurant profile.
- As a restaurant owner, I want to edit restaurant information.
- As a restaurant owner, I want to view all reviews.
- As a restaurant owner, I want to reply to reviews.
- As a restaurant owner, I want to see average ratings.
- As a restaurant owner, I want to delete my restaurant profile.
- As a restaurant owner, I want to logout safely.
